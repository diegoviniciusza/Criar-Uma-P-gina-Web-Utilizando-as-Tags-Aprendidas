O que é HTML

HTML, ou Linguagem de Marcação de Hipertexto, é a linguagem padrão usada para criar e estruturar páginas web. É a base para a construção de sites, permitindo a inclusão de textos, imagens, links e outros elementos. HTML não é uma linguagem de programação, mas sim uma linguagem de marcação, o que significa que ela usa tags para estruturar o conteúdo e definir como ele deve ser exibido no navegador.

  • Tag de Cabeçalho - h1

    • No HTML, as tags de cabeçalho (headings) são usadas para definir títulos e subtítulos em uma página web. Elas variam de < h1> a < h6>, sendo < h1> o título mais importante e < h6> o menos importante.

  • Tag de Paragrafo - p

    • A tag de parágrafo em HTML é simplesmente a tag < p>. Ela é usada para agrupar e exibir blocos de texto em um documento HTML, ou seja, serve para demarcar um parágrafo de texto comum.

  • Tag de Marca Texto - mark

    • A tag < mark> em HTML serve para destacar ou realçar um trecho de texto dentro de um parágrafo ou de qualquer outro elemento de texto. Pense nela como se você estivesse usando um marca-texto amarelo em um livro para chamar a atenção para algo importante.

  • small

    • A tag < small> em HTML é usada para representar um texto pequeno, geralmente para comentários de rodapé, notas de direitos autorais, letras miúdas ou outras observações secundárias. Pense nela como a fonte que você encontra em uma nota de rodapé de um contrato ou no rodapé de um site, que não é o conteúdo principal, mas é importante para a informação completa.

  • i

    • A tag de itálico em HTML tem duas formas principais: < i> e < em>. Embora ambas as tags façam o texto aparecer em itálico por padrão, elas têm significados semânticos diferentes.

  • u

    • A tag < u> em HTML é usada para sublinhar um trecho de texto. Basicamente, ela coloca uma linha embaixo das palavras ou frases que você envolve com ela.

  • strong

    • A tag < strong> em HTML é usada para indicar que um trecho de texto tem forte importância, urgência ou relevância. Por padrão, os navegadores exibem o conteúdo dentro da tag < strong> em negrito.

  • ul

    • A tag

      em HTML é usada para criar uma lista não ordenada. Pense nela como uma lista de itens onde a ordem não importa, como uma lista de compras ou uma lista de características. Cada item dentro dessa lista é marcado por um "marcador" (geralmente um ponto, bolinha ou quadrado), não por números.
